Introduction

First, Nous allons voir comment allumer des leds avec un petit montage arduino. Puis nous nous servirons de ça pour faire du code morse. Pour ce tutoriel nous allons avoir besoin d’une carte Arduino Uno, de câble et d’une led rose (ou n’importe quel couleur, mais rose c’est plus jolie)

Physical assembly

Avant de faire le code, il va falloir faire le montage physique ! On va brancher la borne + de la led sur un pin de l’arduino, nous on prend le 2. On va brancher le - sur le ground. On ajoute une résistance pour protéger la led (acheter une nouvelle led chaque jour ça va vite être coûteux !)

Pour le montage physique, c’est tout pour le moment. On a pas besoin du cablage d’un pc de la NASA non plus !

Arduino code

Passons à la partie la plus fun ! Le code ! Tout d'abord on va mettre dans setup certains trucs importants pour que le code marche après *faire des screen du code dans vsc* 


void setup()
{
  Serial.begin(9600);
  pinMode(2, OUTPUT);
}

Puis dans loop on va mettre le code à exécuter. 


void loop()

Cette ligne va “appeler” la fonction waitInput qui sert à attendre que l’utilisateur écrit un truc puis appuis sur entrer. 


String waitInput(String msg)
{
  Serial.println(msg);
  while(Serial.available()<=0){}
  delay(100);
  String userInput="";
  while(Serial.available()>0)
  {
    userInput+=Serial.read();
  }
  return String(userInput);
}
